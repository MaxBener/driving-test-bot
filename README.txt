Инструкция по запуску на Render:

1. Залей этот проект на GitHub.
2. Перейди на https://dashboard.render.com/
3. Нажми "New +" → "Web Service" → Подключи GitHub
4. Выбери репозиторий с этим проектом.
5. Render автоматически обнаружит файл render.yaml и создаст воркер.
6. После запуска бот будет работать 24/7 и слать уведомления в Telegram.
